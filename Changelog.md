# Version 1.1.0 - 2/17/2026

## New Features:

Navigation Drawer: Completely replaced the legacy tab system and bottom buttons with a modern slide-out navigation drawer.

Settings Menu: Introduced a dedicated Settings screen for app-wide configurations.

Dark Mode Persistence: Added a Dark Mode toggle that saves your preference to device storage, remembering your choice between sessions.

## UI/UX Improvements:

Maximized Viewport: The removal of static navigation bars allows the HTML tools (Character Sheet, Generators) to utilize the full screen height.

Floating Menu: Added a discrete menu button in the bottom-left corner to access navigation without obstructing interface elements.

External Link Indicators: Added visual cues to the "Donate" button to indicate it opens in an external browser.

## Miscellaneous:

Dynamic Versioning: The About section now pulls version information directly from internal asset files. (Makes my life easier)

Documentation: Added a direct link to the Changelog within the About section.

Code Cleanup: Optimized imports and suppressed unused function warnings for cleaner maintenance.
